「打球速度174km/h」「バックスピン2300rpm」——かつては肉眼でしか評価できなかった打撃・投球の数字が、今やリアルタイムでテレビ画面に表示される時代になった。これを可能にしたのが**Statcast**だ。

## Statcastとは

StatcastはMLBが2015年から全30球場に導入した、カメラとレーダーを組み合わせたボール・選手トラッキングシステムだ。

**主なハードウェア:**
- **ホークアイカメラ** — 12台以上の超高速カメラが打球・走路・守備位置をミリ秒単位で追跡
- **ドップラーレーダー** — 投球・打球の速度と軌道を計測

これらが組み合わさることで、従来の「スコアブック」では記録できなかったあらゆる動きが数値化される。

## Statcastが計測・生成する主要指標

### 打撃系
| 指標 | 内容 |
|------|------|
| **Exit Velocity（打球速度）** | バットにボールが当たった瞬間の速度（mph/km/h） |
| **Launch Angle（発射角）** | 打球の打ち出し角度（度） |
| **Expected BA（xBA）** | 打球速度と角度から算出した「期待打率」 |
| **Expected SLG（xSLG）** | 同様に算出した「期待長打率」 |
| **Barrel（バレル）** | 本塁打・長打になる確率が高い最高品質の打球 |

### 投球系
| 指標 | 内容 |
|------|------|
| **Spin Rate（回転数）** | 投球の回転数（rpm）。カーブ・スライダーの切れに直結 |
| **Release Point** | リリースポイントの一貫性を計測 |
| **Induced Vertical Break** | バックスピンによる「ホップ」の量 |
| **Extension** | マウンドからホームベースに向かう到達距離（リリース位置） |

### 走塁・守備系
| 指標 | 内容 |
|------|------|
| **Sprint Speed** | 最高走速（mph/km/h） |
| **Jump** | 外野手が打球に対して最初に反応する速さ・方向 |
| **Outs Above Average（OAA）** | 守備機会の難易度を加味した守備貢献 |

## Statcastが変えた「野球の見方」

### 1. 「あたり損ね」を再定義した

かつて「力のない当たり」に見えた打球でも、打球速度と角度が最適なら本塁打になる。「バレル打球（Barrel）」の概念は、目に見える結果ではなく「打球の質」で打者を評価することを可能にした。

### 2. 高速ストレートの「体感速度」が数字に

同じ95mph（約153km/h）の速球でも、バックスピン量（Spin Rate）が多い投手の球は「ホップしているように見える」。これが打者に体感的に「速く感じる」原因だ。山本由伸のフォークの急激な落下もSpin Rateで説明できる。

### 3. 守備を客観評価できるようになった

「あのショートは守備範囲が広い」という主観的評価が、Outs Above Average（OAA）という数字で検証できるようになった。守備はセイバーメトリクスで最も指標化が難しい領域だったが、Statcastが突破口を開いた。

## NPBのStatcast相当データ

NPBでは「DELTA（データスタジアム社）」や「Trackman」システムが一部球場で導入され、日本プロ野球でも打球速度・回転数データが活用されつつある。ただしMLBほどの全球場・全試合での完全公開には至っていない。

Statcastは「目に見えない野球の本質」をデータで可視化した。これを理解することで、試合観戦の解像度が劇的に上がる。

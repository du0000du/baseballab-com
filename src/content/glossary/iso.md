**ISO（Isolated Power / 純長打率）= 長打率 − 打率**

計算は引き算だけ。長打率から打率を引くことで、単打の貢献分を排除した「長打だけ」の価値が残る。

## ISOの目安

| ISO | 評価 |
|-----|------|
| .100未満 | 長打力は低い |
| .100〜.149 | 平均以下 |
| .150〜.199 | MLB平均 |
| .200〜.249 | 長打力が高い |
| .250以上 | エリートの長距離打者 |
| .300以上 | 歴史的なパワーヒッター |

ジャッジの2022年シーズン（本塁打62本）のISOは **.357**。このレベルに達するのは、MLBの長い歴史でも両手で数えられるほどしかいない。ISOはシンプルだが、「スラッガーとアベレージヒッターを一目で区別したい」ときには十分な指標だ。

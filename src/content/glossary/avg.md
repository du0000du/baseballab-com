「打率3割」という言葉は、プロ野球中継でも日常会話でも当たり前のように使われる。3割を超えた選手は一流、2割5分以下は苦しい——そんなイメージが定着している。しかし「打率とは何か」を正確に理解している人は意外と少ない。

## 打率の計算式

```
AVG（打率）= 安打数 ÷ 打数
```

**打数**は「打席数」とは異なる。四球・死球・犠打・犠飛は打数にカウントされない。つまり「ヒットを打てる可能性のある場面」で、「実際にヒットを打った割合」が打率だ。

### 計算例

| 場面 | 安打 | 打数 |
|------|------|------|
| シングルヒット | +1 | +1 |
| 三振 | 0 | +1 |
| 四球 | 0 | **カウントしない** |
| 犠飛 | 0 | **カウントしない** |

100打数で30安打なら打率.300（3割）。

## 打率の目安（MLB基準）

| AVG | 評価 |
|-----|------|
| .320以上 | 打撃王争い圏内。首位打者候補 |
| .300〜.319 | 優秀。レギュラー一流クラス |
| .270〜.299 | 平均〜やや上 |
| .250〜.269 | 平均的なレギュラー |
| .230未満 | 守備・走塁での貢献が求められる |

NPBでは全体的にやや低め（MLBより難度が高いとされる）で、.280前後が平均レギュラーの目安になる。

## 打率の「見えない限界」

打率が高くても評価が分かれる理由がある。

**四球を稼ぐ選手が不当に低評価される**  
イチローはキャリア通算打率.311を誇る伝説的打者だったが、四球を選ぶよりヒットを狙うスタイルだった。一方、バリー・ボンズはピーク時に「歩かせた方がマシ」と相手に思わせるほど出塁能力が高く、打率.370でもOBP（出塁率）が.609に達した年がある。四球を稼ぐ能力は打率に現れない。

**長打力が見えない**  
シングルヒット1本もホームラン1本も、打率計算上は同じ「1」だ。10本塁打打者と50本塁打打者が同じ.280でも、チームへの貢献は全く異なる。

**BABIP（インプレー打率）の影響**  
どんな打者でも運次第でボールがフィールドに飛んだとき安打になるかアウトになるかが変わる。打率は守備の良さや運にも影響される。

## 現代野球では「3指標セット」で評価する

```
AVG（打率）→ どれだけ当てるか
OBP（出塁率）→ どれだけ塁に出るか
SLG（長打率）→ どれだけ遠くに飛ばすか
```

この3つを一度に見ることで、打者の全体像が掴める。さらにOPS（出塁率＋長打率）やwOBAを使えば、1つの数字で比較できる。

打率は野球入門の第一歩として依然重要な指標だが、それだけで打者を評価するのは限界がある——それが現代セイバーメトリクスの出発点だ。
